参数提取节点

参数提取节点是腾讯云智能体开发平台工作流中的一个重要组件，用于从用户对话中提取关键信息（参数）。

节点功能

参数提取节点属于信息收集类节点，通过与用户进行多轮对话来收集参数。这些参数可以是名词、地名、手机号码、医院名称等各种业务相关信息。工作流执行至该节点时会中断并与用户交互，直到收集到必需的参数。

操作说明

输入变量

输入变量仅在该节点内部生效，不能跨节点使用。支持最多添加50个输入变量，满足多输入变量场景需要。

配置项包括：

  • 变量名称：变量的名称，只能包含字母、数字或下划线，并且以字母或下划线开头（必填）
  • 变量描述：变量的说明信息（非必填）
  • 数据来源：变量的数据来源，支持"引用"和"输入"两种选项
  • 变量类型：变量的数据类型，默认为"引用"的变量类型或"输入"的string类型

系统默认会传入三个变量：本轮对话内容、对话历史和当前时间，无需手动添加。

收集参数

模型设置

您可以设置该节点所使用的大语言模型来从与用户对话中提取所需参数，仅支持选择该账号具有权限的大模型。

参数信息配置

参数信息配置包括：

  • 参数名称：参数的名称，建议使用具体的中文名称，便于模型识别理解（必填）
  • 参数类型：参数的数据类型（必选）
  • 参数描述：参数的说明信息，帮助模型精准识别理解参数（必填）
  • 参数收集示例：参数期望的正确示例和错误示例（可选）
  • 是否必选：参数是否必填

提示词配置

如果您对参数提取过程有特殊要求，可以配置提示词来引导大模型生成相应的反问澄清话术。

典型场景包括：

  1. 对话中未提取到参数时，告知模型如何反问用户
  2. 回复内容有特定的格式要求或偏好

输出变量

经该节点处理后的输出变量默认为用户配置的参数，以及运行时报错信息 Error（数据类型为 object，正常运行时该字段为空）。不支持用户手动新增。

常见问题

如果您希望复制其他参数的配置信息，可以单击新增参数框右上角的"复制"图标，选择您需要的参数进行复制。
